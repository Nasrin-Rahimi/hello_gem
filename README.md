Hello_World

This Simple Command Line Interface(CLI), just have a "Hello_World" message. The porpuse of this simple code, is to create and publish a Gem.

Installation
    $ gem install hello

Language
    Ruby

Usage
    $ irb
    >> require 'hello'
    => true
    >> Hello.hi
    Hello world!
